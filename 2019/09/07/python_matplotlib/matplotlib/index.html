随机漫步

随机漫步每次行走都完全是随机的，没有明确的方向，结果是由一系列随机决策决定的。

创建 RandomWalk() 类

为模拟随机漫步，我们将创建一个 RandomWalk 类，它随机的选择前进方向。这个类需要三个属性，其中一个是存储随机漫步次数的变量，另外两个是列表，分别存储随机漫步经过的每个点的 x 和 y 坐标。
